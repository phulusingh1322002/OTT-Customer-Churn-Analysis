# OTT-Customer-Churn-Analysis

## Overview
This project performs customer churn analysis using Python, SQLite, Pandas, NumPy, Matplotlib, and Seaborn. It demonstrates data cleaning, feature engineering, data analysis, and visualization to identify customer churn patterns and business insights.

## Features
- Import data from SQLite database
- Clean and preprocess customer data
- Handle missing values and data type conversions
- Create churn-related features
- Merge multiple datasets
- Calculate key business KPIs
- Visualize churn trends and customer behavior
- Export cleaned dataset to CSV

## Technologies Used
- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
- Seaborn

## Dataset
The project uses three tables:
- **db_customer** – Customer information
- **db_subscription** – Subscription and churn details
- **db_support** – Customer support records

## Key Analysis
- Churn Rate
- Retention Rate
- Churn by Plan Type
- Average Revenue Per User (ARPU)
- Average Customer Tenure
- Revenue at Risk
- Escalation Rate
- Average Complaints per User
- Correlation between Escalations and Churn
- Customer Churn Risk Classification

## Visualizations
- Monthly Churn Trend
- Churn Rate by Plan Type
- Churn Rate by State
- Correlation Heatmap
- Pair Plot
- Categorical Comparison Charts

## Output
- Cleaned and merged customer dataset
- Exported CSV file (`exported_churn_data.csv`)
- Business insights with charts and KPI calculations
